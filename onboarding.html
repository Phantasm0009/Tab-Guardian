Password Protection

Lock any tab with a password to keep sensitive information private.

  1. Click the Tab Guardian icon in your browser toolbar
  2. Set a global password in Settings (you only need to do this once)
  3. Click "Lock This Tab" on any page you want to protect
  4. Your tab is now protected and will require a password to access

Pro Tip: When unlocking a tab, check "Unlock domain on all tabs" to unlock all tabs from the same website.

Auto Declutter

Keep your browser running smoothly by automatically closing unused tabs.

  1. Tab Guardian tracks how long it's been since you've used each tab
  2. Set your preferred inactive time threshold in Settings
  3. Inactive tabs will be automatically closed and saved to the Tab Vault
  4. You can whitelist important sites that should never be auto-closed

Pro Tip: Use "Clean Up Now" to manually declutter tabs when your browser feels sluggish.

Tab Vault

Never lose an important tab again with the Tab Vault history feature.

  1. All closed tabs are automatically saved to the Tab Vault
  2. Click "View Tab Vault" to see your tab history
  3. Filter tabs by how they were closed (manual, auto-decluttered, etc.)
  4. Easily reopen any tab with a single click

Pro Tip: The Tab Vault makes it safe to be aggressive with tab decluttering since you can always recover closed tabs.

Stealth Mode

Quickly hide tab contents with stealth mode for instant privacy.

  1. Click "Stealth Mode" in the Tab Guardian popup
  2. Enter a custom title to disguise your tab
  3. The tab's title and favicon will be replaced with innocent alternatives
  4. Perfect for quickly hiding sensitive content when someone walks by

Pro Tip: You can set a default stealth title in the extension settings.

Keyboard Shortcuts

Tab Guardian supports keyboard shortcuts for quick access to commonly used features.

Alt+L
Lock current tab
Alt+U
Unlock all tabs
Alt+S
Toggle stealth mode

Note: You can customize these shortcuts in Chrome's extension keyboard shortcut settings.
